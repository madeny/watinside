# Welcome to your new site!

This site is built using the **Minimal Mistakes** theme, one of the most popular and customizable themes for Jekyll.

## Getting Started

1.  **Configure**: Edit `_config.yml` to change settings, add your social links, and enable analytics.
2.  **Content**: Add new posts to `_posts/` or create new pages (like this one).
3.  **Navigation**: Edit `_data/navigation.yml` to change the top navigation links.
4.  **Styles**: You can change the skin in `_config.yml` (e.g., `minimal_mistakes_skin: dark`).

Check out the [Minimal Mistakes documentation](https://mmistakes.github.io/minimal-mistakes/docs/quick-start-guide/) for all the details.
